# quillio
Source code for the quillio app which can be found at http://quillio.ink

## backend
In order to build the backend:

```bash
git clone git@github.com:ismailmustafa/quillio.git
cd quillio/
stack build
stack exec quillio
```
It will run on port 80.

## frontend
```bash
git clone git@github.com:ismailmustafa/quillio.git
cd quillio/frontend/
grunt serve
```
