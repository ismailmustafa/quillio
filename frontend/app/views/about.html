What is it?

Quillio (Quill + Twilio) is a project born out of a desire to send handwritten messages to my girlfriend in the laziest way possible. It enables you to send beautifully handwritten images as text messages.

Technologies used

Backend
  • Servant
  • Haskell Handwriting.io API Wrapper
  • Amazon EC2
Frontend
  • AngularJS
  • Materialize
  • Yeoman
Third party APIs
  • handwriting.io
  • Twilio
