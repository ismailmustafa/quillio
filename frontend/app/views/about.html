What is it?

Quillio (Quill + Twilio) is a project born out of a desire to send handwritten messages to my girlfriend in the laziest way possible. It enables you to send beautifully handwritten images as text messages.

Technologies used

Quillio uses handwriting.io to generate a realistic handwritten image that is then sent as a message to any phone number using Twilio. The backend is built using Servant, and the frontend is built using AngularJS and Materialize.
